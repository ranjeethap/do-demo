# Overview

This repository provides a complete demo environment to showcase:
- DOKS-based microservice deployment
- Ingress via nginx + `sslip.io`
- GitHub Actions CI/CD integration
- Horizontal Pod Autoscaling (HPA)
- Canary strategies (Flagger-ready)
- DOCR-based image storage and secure pulls
